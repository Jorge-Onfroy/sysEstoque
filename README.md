Sistema de Estoque

Sistema simples de controle de estoque desenvolvido com PHP, MySQL e Bootstrap.

📌 Sobre o projeto

Este sistema foi criado com foco em aprendizado e prática de desenvolvimento web, incluindo autenticação de usuários, controle de sessão e CRUD de produtos.



⚙️ Funcionalidades

- Login de usuários
- Sistema de sessão (auth)
- Controle de permissões (admin e usuário)
- Cadastro de produtos
- Listagem de produtos
- Exclusão de produtos (somente admin)
- Status automático de estoque



🧑‍💻 Níveis de acesso

=== Admin ===
- Criar, editar e excluir produtos
- Acesso total ao sistema

=== Usuário ===
- Visualizar produtos
- Sem permissão para excluir



🛠️ Tecnologias usadas

- PHP
- MySQL
- Bootstrap 5
